# Directory List and Print Pro

## Overview

Directory List and Print Pro is a software tool designed for Windows users to create detailed lists of files and directories. It helps organize, view, and export directory contents in various formats, supporting tasks related to file management and system auditing.

## Key Features

**Generate comprehensive directory listings including subfolders**  
**Export file and directory lists to multiple formats such as TXT, CSV, and XLS**  
**Filter files by extension, size, or date to customize output**  
**Include file properties like size, date modified, and attributes in reports**  
**Preview directory contents before exporting**  
**Save and load user-defined settings for repetitive tasks**  
**Support for printing directory lists directly from the application**  
**User-friendly interface optimized for clarity and ease of use**
